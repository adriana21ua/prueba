Ejercico asigando

Problema al resolver

Desarrolle una aplicación web para resolver el siguiente problema de Kanas Tours
Diseñe un formulario para capturar los siguientes datos:

1- Datos del cliente (nombre, apellido, entradas)
2- Fecha del viaje.
3- Datos del viaje.
4- Cantidad de entradas a comprar.
5- Forma de pago (efectivo o crédito).
1- Monto inicial, descuento y total a pagar.
2- Si el cliente utiliza la forma de pago crédito, se le realizará un cargo del 5%.

Analisis del problema

Caja negra

Entrada de datos

Datos del cliente
1. Nombre
2.Apellido
3.Entradas
4.Fecha
5.Destino
6.Precio
8.Forma de pago

Proceso

Cálcular el monto inicial:
1.Monto inicial = Precio por entrada , Cantidad de entradas
2.Cálculo del descuento.
3.Cálculo del total a pagar.
Si forma de pago es crédito
Recargo = 5% del monto inicial.
Total final = Monto inicial - Descuento + Recargo.
Si forma de pago es efectivo:
Total final = Monto inicial igual a Descuento.

Salida de datos

Datos del cliente
Fecha del viaje
Destino
Cantidad
Precio
Subtotal
Descuento
Recargo
Total final
